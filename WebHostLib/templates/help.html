Help Center

Using SekaiLink

Guides for getting started, playing with others, and troubleshooting common issues.

Getting Started

  • Downloads are coming soon. See Downloads for status.
  • Import your ROM or game files into SekaiLink.
  • Configure your game settings and click Launch.

Solo Play

  • Import a game, configure settings, and generate a seed.
  • SekaiLink patches, launches the emulator, and connects automatically.
  • No account or internet connection required for solo play.

Multiplayer Rooms

  • Create or join a room from the Room List.
  • Each player configures their own game settings.
  • The host generates when everyone is ready.

Generation

  • Host clicks Generate once everyone is ready.
  • Generation locks settings until completion.
  • SekaiLink distributes the results to all players automatically.

Room Management

  • The host can manage players in a room.
  • Host can transfer room ownership to another player.
  • Room codes and shareable links make joining straightforward.

Troubleshooting

  • Check the connection status indicators in SekaiLink.
  • Use the "Copy Debug Info" button to collect diagnostic data.
  • See the Docs for common issues and solutions.

Need help?

The fastest way to get support is through the community Discord. You can also file an issue on GitHub for confirmed bugs or feature requests.
