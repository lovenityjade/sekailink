## Bugs corrected

- Restored the missing Linux Canonical `20260704` client archive on the CDN.
  The public URL now serves the expected ZIP instead of the website fallback,
  and its SHA-256 matches the active release manifest. Affected platforms:
  Linux Canonical.
- Constrained the ALttP `boss_shuffle` option to the choices supplied by Nexus
  instead of rendering it as unrestricted text. Affected games: A Link to the
  Past.
- Made Discord bug-report attachment paths include the Discord attachment ID,
  preventing same-named screenshots in one message from colliding during
  administrative export. Affected surface: social-bots bug export.
- Made the AP patch wrapper apply the ROM selected for launch to the exact
  FireRed or LeafGreen AP setting selected by the patch extension, instead of
  allowing a stale persisted path to win. The shared module now requires both
  base ROMs and selects the matching one for `.apfirered` or `.apleafgreen`.
  Affected games: Pokemon FireRed and LeafGreen.
- Fixed the ALttP stuck-item recovery path so it rewinds the ROM receive index
  before clearing and retrying the occupied slot. The previous recovery skipped
  the item permanently after advancing the index and could not recover a stuck
  final item. Affected games: A Link to the Past.
- Removed the obsolete generic libretro Ocarina of Time module from runtime
  availability checks; the supported path is the managed Ship of Harkinian
  package. Affected games: Ocarina of Time.
