# SekaiLink Compatibility Tiers

SekaiLink needs trailer-friendly claims, but those claims need a technical floor.
Use these tiers for public and internal language.

| Tier | Label | Meaning |
|---|---|---|
| 0 | Researched | APWorld/tracker resources have been identified, but no local runtime proof exists. |
| 1 | Indexed | Game exists in the local registry with enough metadata to plan integration. |
| 2 | Tracker Ready | Tracker pack or bundle loads in Sekaiemu/SKLMI, but AP runtime sync is not proven. |
| 3 | Runtime Heartbeat | Game boots and at least one check or item receive path is proven through SKLMI/AP. |
| 4 | Playable Candidate | Multiple checks/items are proven, room sync works, and known limitations are documented. |
| 5 | Release Certified | Windows/Linux client packaging, room server sync, update path, CDN artifacts, and docs are verified. |

## Public Language

- Tier 0-1: "Research ready" or "planned support".
- Tier 2: "Tracker preview".
- Tier 3: "Compatibility preview" or "runtime heartbeat".
- Tier 4: "Playable candidate".
- Tier 5: "SekaiLink compatible" or "release certified".

Do not use "SekaiLink compatible" for a game below Tier 5 unless the trailer or
post clearly says "preview".
