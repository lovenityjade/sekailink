# Future Hint Quest Note

A requested hint may later create a social quest for the player who owns the hinted item.
If that player collects the item naturally, without release, cheat, or admin grant, they may receive a cumulative reward.

This is intentionally out of scope for the first Client Core runtime-social pass. The current implementation only moves chat, activity, hint selection, pending items, and full notification surfaces out of Sekaiemu and into Client Core.
