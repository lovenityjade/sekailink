# Platform Sync Policy

Every change record must mention Linux and Windows.

## Required fields

- Linux impact:
- Windows impact:
- Build needed:
- Runtime bundle impact:
- Release gate:

## Rule

Linux/Windows synchronization is required before a public release when the change
touches Client Core, Sekaiemu, runtime packaging, SKLMI launch, updater or
client signals.

For this Core Access plan, synchronization is documented but not implemented.
