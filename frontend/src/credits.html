Archipelago Multiworld Randomizer

The Foundation of Everything

SekaiLink is built on top of Archipelago, the incredible open-source multiworld randomizer that makes all of this possible. We use their generation tools, server software, and game implementations.

Archipelago is maintained by a dedicated team of developers and hundreds of contributors who create game worlds, fix bugs, and build features for the community.

Visit Archipelago.gg GitHub Repository
Important: SekaiLink is NOT officially affiliated with Archipelago. We're an independent platform that uses their open-source code under the MIT license.

Racetime.gg

Architectural Inspiration

SekaiLink's architecture and design patterns were heavily inspired by racetime.gg, the premier platform for speedrunning races. Their state machine approach, lobby management system, and permission hierarchy served as excellent references for building SekaiLink.

Thank you to the racetime.gg team for building such a well-architected platform and making it open source!
