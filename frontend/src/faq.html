General

What is SekaiLink?

SekaiLink is a hosting platform for Archipelago multiworld sessions. We provide YAML management, ROM verification, automatic seed generation, and server hosting to make playing Archipelago syncs easier.

Is SekaiLink free?

Yes! SekaiLink is completely free to use. We accept donations to help cover server costs, but all features are available to everyone at no charge.

Is SekaiLink affiliated with Archipelago?

No, SekaiLink is an independent project. We use Archipelago's open-source code and tools, but we are not officially affiliated. Do NOT ask for SekaiLink support on the Archipelago Discord - use our Discord server instead.

How many games are supported?

SekaiLink supports all 89+ officially supported Archipelago games, plus approved custom worlds. You can view the complete game list on the homepage.

Setup & Getting Started

How do I create an account?

SekaiLink uses Discord OAuth for authentication. Simply click "Sign in with Discord" on the homepage and authorize the application. No separate account creation is needed.

Do I need to upload my YAML every time?

No! Once you upload or create a YAML, it's saved to your account. You can reuse it in multiple lobbies. Go to Dashboard → YAML Manager to see all your saved YAMLs.

Which games require ROM uploads?

Games marked with a ROM Required badge need ROM files for generation. Common examples include:

  • The Legend of Zelda: A Link to the Past
  • Super Metroid
  • Super Mario World
  • Final Fantasy
  • Pokémon Red/Blue

Check the game page or your ROM Manager to see which games you need ROMs for.

Where can I get ROMs?

SekaiLink does NOT provide ROMs. You must obtain your own legally dumped ROMs. We cannot provide links or assistance in acquiring ROMs due to legal restrictions.

My ROM upload failed. What should I do?

ROM verification can fail for several reasons:

  • Wrong version - Make sure you're using the correct region (usually USA or Japan)
  • Pre-patched ROM - ROMs must be unmodified/vanilla
  • Incorrect game - Double-check you selected the right game

The error message will show the expected SHA-1 hash. You can verify your ROM's hash using tools like HashCalc or 7-Zip.

Lobbies & Playing

How do I create a lobby?

Click "Create Lobby" from the homepage or any game page. Fill out the lobby settings (name, player limit, rules, etc.) and click "Create Lobby". You'll be taken to the lobby page where players can join.

Can I join a lobby after generation starts?

No, you must join before the host starts generation. Once generation begins, the lobby is locked and no new players can join.

What happens if the host leaves?

If the host leaves before generation, another player will be promoted to host automatically. If the host leaves after the server is running, the server continues and other players can still play. Moderators can also take control of lobbies if needed.

How long does generation take?

Generation typically takes 1-5 minutes depending on:

  • Number of players (more players = longer generation)
  • Game complexity (some games take longer to generate)
  • Server load (peak times may be slower)

You'll see real-time progress updates in the lobby chat during generation.

What does "DNF" and "Forfeit" mean?
  • DNF (Did Not Finish) - You can't complete the sync but want to stay connected to help others
  • Forfeit - You're leaving the sync entirely and disconnecting

Using these properly instead of rage quitting helps maintain good server ratings and etiquette.

Can I play multiple games in one lobby?

Yes, if the host enabled "Allow Multi-game" when creating the lobby. This lets each player join with different games. If disabled, all players must play the same game.

Technical Issues

I can't connect to the server. What should I do?

Try these troubleshooting steps:

  1. Verify the server address and port in the lobby page
  2. Make sure the lobby state is "Active" (server must be running)
  3. Check your firewall settings - ensure the port isn't blocked
  4. Restart your Archipelago client
  5. Try connecting using the IP address instead of hostname

If still unable to connect, ask for help in our Discord server.

Generation failed. What went wrong?

Common causes of generation failure:

  • Invalid YAML - One or more players have malformed settings
  • Missing ROM - A required ROM wasn't uploaded
  • Incompatible settings - Player settings create an impossible seed
  • Custom world error - An unstable custom world caused a crash

Check the lobby chat for specific error messages. The host can try generation again after fixing issues.

Is there a time limit?

Lobbies can have optional time limits (1-24 hours) set by the host. If "Restrict Time Limit" is enabled, the server will automatically release all items and close the lobby when time expires.

If not restricted, the time limit is just a guideline and won't force anything.

Are ROMs stored permanently?

No. All ROMs are automatically deleted on the 1st of each month for storage and legal reasons. You'll need to re-upload them monthly if you want to play ROM-based games.

Can I download other players' ROMs?

Absolutely not. ROMs are used ONLY for seed generation on the server. There is no way to download ROMs, period. Don't ask for this feature - it will never be added.

Account & Privacy

How do I delete my account?

Go to Dashboard → Profile & Settings and click "Delete Account". This will permanently delete your account, YAMLs, and all associated data.

Warning: This action cannot be undone!

What data does SekaiLink collect?

We collect:

  • Discord user ID, username, and avatar (from OAuth)
  • Email address (optional, for important notifications only)
  • YAMLs you create/upload
  • ROMs you upload (automatically deleted monthly)
  • Lobby participation history
  • Chat messages (for moderation purposes)

We never share your data with third parties. See our About page for more details.

What are ratings and reviews?

After playing in a lobby, you can rate other players on 4 criteria (punctuality, respect, following rules, valid release). These ratings help build a community of good players.

You can also write reviews, which are moderated before appearing on profiles.

Can I get banned?

Yes. Users can be banned for:

  • Harassment or toxic behavior
  • Repeated rule violations
  • Cheating or exploiting
  • Spamming or advertising

Bans can be temporary or permanent. You can appeal bans via the Contact page.
