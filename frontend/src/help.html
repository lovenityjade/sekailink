Getting Started

What is SekaiLink?

SekaiLink is a hosting platform for Archipelago multiworld sessions. We handle YAML collection, ROM verification, seed generation, and server hosting so you can focus on playing.

Quick Start Guide

  1. Sign in with Discord - Click the "Sign in with Discord" button on the homepage
  2. Create or upload your YAML - Go to Dashboard → YAML Manager
  3. Upload ROMs (if needed) - Some games require ROM files for generation
  4. Join or create a lobby - Browse active lobbies or create your own
  5. Wait for generation - Once all players are ready, the host starts generation
  6. Download your patch and play! - Connect to the server and enjoy
Important: Archipelago.gg and their community do NOT provide technical support for SekaiLink. For issues with SekaiLink, please visit our Discord server.

Creating YAMLs

What is a YAML?

A YAML file is your personal configuration for a game in Archipelago. It defines your player name, game settings, item randomization options, and more.

Creating a YAML on SekaiLink

  1. Go to Dashboard → YAML Manager
  2. Click "Create YAML"
  3. Select your game from the dropdown
  4. Fill out the settings form with your preferences
  5. Click "Save YAML"

Uploading an Existing YAML

If you already have a YAML file from the Archipelago website or generator:

  1. Go to Dashboard → YAML Manager
  2. Click "Upload YAML"
  3. Select your .yaml or .yml file
  4. The system will validate your file automatically
Common YAML Mistakes:
  • Missing or incorrect name field
  • Invalid game identifier
  • Incorrect YAML syntax (indentation matters!)

Example YAML Structure

name: YourPlayerName
game: A Link to the Past
A Link to the Past:
  progression_balancing: 50
  accessibility: items
  # ... more settings ...

ROM Management

Why ROMs Are Needed

Some games (like A Link to the Past, Super Metroid, etc.) require ROM files to generate patches. These ROMs are used ONLY for generation and are never downloadable by other users.

Legal Notice: You must provide your own legally obtained ROM files. SekaiLink does not provide ROM downloads. All ROMs are automatically deleted on the 1st of each month.

Uploading ROMs

  1. Go to Dashboard → ROM Manager
  2. Find the game you need to upload a ROM for
  3. Click "Upload ROM"
  4. Select your ROM file
  5. The system will verify the ROM's SHA-1 hash automatically

ROM Verification

ROMs are verified using SHA-1 hashes to ensure they're the correct version for Archipelago. If your ROM fails verification:

  • Make sure it's the correct region (usually USA/Japan)
  • Ensure it's not a pre-patched or modified ROM
  • Check that it's the correct game version

Creating Lobbies

How to Create a Lobby

  1. Click "Create Lobby" from the homepage or game page
  2. Enter a lobby name and description
  3. Set player limit (2-20 players)
  4. Configure lobby settings (see below)
  5. Click "Create Lobby"

Lobby Settings

Setting Description
Time Limit Maximum time allowed for the sync (1-24 hours, or unlimited)
Restrict Time Limit When enabled, items will auto-release and lobby closes when time expires
Allow Multi-game Players can join with different games
Disallow ROM Games Only allow games that don't require ROM uploads
Disallow Custom Worlds Only allow officially supported Archipelago games
Allow Broadcasting Players can mark themselves as streaming
Voice Chat Enabled Enable Discord voice channel for this lobby
Blacklisted Games Select specific games that are not allowed in this lobby

Host Controls

As the lobby host, you have access to:

  • Generate Seed - Start the generation process when all players are ready
  • Start Sync - Begin the timer when everyone is connected
  • Admin Console - Send server commands (/release, /kick, etc.)
  • Close Lobby - Prevent new players from joining
  • Delete Lobby - Permanently delete the lobby

Joining Lobbies

Finding Lobbies

Browse active lobbies from:

  • Homepage - Shows all active lobbies
  • Game pages - Shows lobbies for that specific game
  • Direct invite link from the host

Joining a Lobby

  1. Find a lobby you want to join
  2. Click "Join"
  3. Select your YAML from the dropdown
  4. If the game requires a ROM, ensure you've uploaded it
  5. Click "Ready" when you're prepared

Player States

  • Waiting - Joined but not ready yet
  • Ready - YAML and ROM (if needed) uploaded, ready for generation
  • Active - Currently playing
  • Finished - Completed the sync successfully
  • DNF - Did not finish
  • Forfeit - Left the sync early

Playing a Sync

Connecting to the Server

  1. Wait for the host to generate the seed
  2. Download your patch file from the lobby page
  3. Apply the patch to your ROM using the appropriate patcher
  4. Launch your Archipelago client
  5. Connect to the server using the address shown in the lobby (sekailink.xyz:PORT)

During the Sync

  • Use the lobby chat to communicate with other players
  • Watch the timer to track elapsed time
  • Mark yourself as "Finished" when you complete your goal
  • Use "DNF" if you need to quit but want to stay connected
  • Use "Forfeit" if you need to leave entirely

Rules & Etiquette

  • Be respectful to other players
  • Don't rage quit - use DNF or Forfeit properly
  • Follow the lobby's custom rules
  • If you need to release items early, ask the host first
  • Report any issues to moderators, not in general chat

Custom Worlds

What are Custom Worlds?

Custom worlds are community-developed game implementations that aren't officially part of Archipelago yet. They may be unstable or have bugs.

Warning: Custom worlds are experimental. If you encounter issues, please report them on the SekaiLink Discord, NOT on the Archipelago Discord.

Uploading Custom Worlds

  1. Obtain a .apworld file from the developer
  2. Go to Dashboard → Custom Worlds
  3. Click "Upload Custom World"
  4. Select your .apworld file
  5. Wait for admin approval (usually 1-3 days)

Once approved, the custom world will be available for use in lobbies.

Troubleshooting

Common Issues

YAML Upload Failed

  • Check your YAML syntax - indentation must be correct
  • Ensure the name field is present
  • Verify the game identifier is valid

ROM Verification Failed

  • Ensure you're using the correct ROM version
  • Check that the ROM isn't pre-patched
  • Verify the region (usually USA or Japan)

Can't Ready Up

  • Make sure you've selected a YAML
  • If the game requires a ROM, ensure it's uploaded and verified
  • Check that the lobby hasn't started generation yet

Generation Failed

  • Check the lobby chat for error messages
  • Verify all players have valid YAMLs
  • Ensure all required ROMs are uploaded
  • If using custom worlds, they may have conflicts

Can't Connect to Server

  • Verify the server address and port are correct
  • Check that the server is running (lobby state should be "Active")
  • Ensure your firewall isn't blocking the connection
  • Try restarting your Archipelago client

Endpoint Method Description

/lobbies/:id GET Get lobby details
/users/:id GET Get user profile
