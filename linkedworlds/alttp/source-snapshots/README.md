# Source Snapshots

Ce dossier preserve les sources actives capturees au moment de la reprise:

- `linkedworld.current.json`
- `sklmi.phase1.current.json`
- docs source utiles

Ces fichiers servent de reference de migration, pas de forme canonique finale.

Regles:

- ne pas modifier ces fichiers pour y poser la verite canonique du repo
- si une information est validee, la recopier proprement dans `manifest/`, `metadata/`, `bridge/`, `tracker/`, `patch/`, `presets/` ou `docs/`
- conserver le nom `*.current.*` pour signaler qu'il s'agit d'une capture d'etat amont
