This is a [PopTracker](https://github.com/black-sliver/PopTracker/) pack for [Ship of Harkinian](https://github.com/HarbourMasters/Archipelago-SoH) with autotracking and other features.

I will expand this README in the future, but for now, you can get the latest release [here.](https://github.com/Brian0255/ship-of-harkinian-ap-tracker/releases/latest)
