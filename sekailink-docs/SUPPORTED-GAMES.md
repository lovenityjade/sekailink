Critere: presence explicite de BizHawk, Dolphin, snes9x-rr ou Project64 dans GAMES-SETUP.md.

Note: le world `tracker` (Universal Tracker) n'est pas un jeu, mais doit etre supporte car il couvre beaucoup de jeux via tracking.
