# Repo Layout

- `server/native/sekailink_server_core`
  - Native Nexus server runtime implementation.
- `deploy/nexus`
  - Nexus-specific systemd units and config templates.
- `third_party/sekailink_runtime/third_party`
  - Header dependency bundle used by the native server build.
- `docs`
  - Scope and operations context for this repository.
  - `nexus-configs.md` documents the Nexus Configs tier and build handoff rules.
